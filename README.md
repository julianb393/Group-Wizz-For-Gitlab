# GitLab-Group-Wizz
Simplify Code Reviews with Group Merge Requests

GitLab Group Wizz is the must-have Chrome extension for developers who want to streamline the code review process within their GitLab groups. This extension allows you to easily view and manage merge requests from people in your selected GitLab group, making collaboration and code reviews faster and more efficient.

### Key Features:
- Group Merge Request Overview: See all the merge requests from members of your selected GitLab group in one place.
- Integrated Code Review Workflow: Quickly access and review code from teammates within your group, helping you stay on top of the latest developments in your projects.
- Simplified Navigation: No need to switch between multiple repositories and pages. The extension gives you a unified, easy-to-use interface to navigate merge requests from across your group.
- Enhanced Collaboration: Improve team collaboration by simplifying the way you track and engage with merge requests within your GitLab group.

### Why Choose GitLab Group Wizz?
* Save Time: The extension aggregates all relevant merge requests from group members, eliminating the need to manually browse different projects and repositories.
* Boost Productivity: Stay focused on reviewing code and collaborating with your team, without the distractions of jumping between different tabs or repositories.
* Improved Code Quality: With easy access to all merge requests within your group, you can ensure timely reviews and smoother project workflows.
* Seamless GitLab Integration: Built to work flawlessly with GitLab, so you can continue using the platform you already know and love.
* Lightweight and Fast: Written entirely in vanilla JavaScript with no external frameworks, providing a responsive experience and full compatibility with Google Chrome.


### Who Should Use This Extension?
- Developers: Spend less time searching for merge requests and more time reviewing code.
- Team Leads: Keep track of merge requests across multiple team members and streamline the review process.
- GitLab Administrators: Simplify project management and enhance collaboration within groups.
